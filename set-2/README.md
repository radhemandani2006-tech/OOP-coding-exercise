# SET-2 (Classes and Objects in JAVA)

## Compilation Guide

To create all class files, run:

```bash
javac *.java
```

---

## Practical 6: Rectangle Class

### Problem Statement:
Define a Java class named Rectangle. It should have two double data fields: width and height, both with a default value of 1. Implement a no-argument constructor and a constructor that takes width and height as parameters. Include methods getArea() and getPerimeter() that return the calculated area and perimeter respectively.

### Sample Output:

```text
Rectangle 1
Width: 1.0
Height: 1.0
Area: 1.0
Perimeter: 4.0

Rectangle 2
Width: 5.0
Height: 10.0
Area: 50.0
Perimeter: 30.0
```

## Practical 8: Point Class

### Problem Statement:
Create a Point class representing a 2D point (x, y). Implement a default constructor that initializes both x and y to 5. Provide a parameterized constructor and a copy constructor. Include a display() method to show the point's coordinates.

### Sample Output:

```text
Default Constructor:
Point Coordinates: (5, 5)

Parameterized Constructor:
Point Coordinates: (10, 20)

Copy Constructor:
Point Coordinates: (10, 20)
```

## Practical 9: Rectangle Comparison

### Problem Statement:
Define a Java class named Rectangle with two double data fields: width and height. Create two Rectangle objects and display their width, height, area, and perimeter. Compare both rectangles based on their areas and print which one has a larger area.

### Sample Output:

```text
Rectangle 1
Width: 4.0
Height: 40.0
Area: 160.0
Perimeter: 88.0

Rectangle 2
Width: 3.5
Height: 35.9
Area: 125.64999999999999
Perimeter: 78.8

Rectangle 1 has a larger area.
```
