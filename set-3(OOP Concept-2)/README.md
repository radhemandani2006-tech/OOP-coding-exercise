# SET-3 (OOP Concepts in JAVA)

## Compilation Guide

To create all class files, run:

```bash
javac *.java
```

## Practical 10: BankAccount Class

### Problem Statement:
Design a class BankAccount with account_holder_name and balance. Use a static variable interest_rate (same for all accounts). Include methods to calculate and display the interest earned. Update interest rate using a static method.

### Sample Output:

```text
Account Holder Name: Radhe
Balance: 50000.0
Interest Rate: 5.0%
Interest Earned: 2500.0

Updating Interest Rate...

Account Holder Name: Radhe
Balance: 50000.0
Interest Rate: 7.5%
Interest Earned: 3750.0
```

## Practical 11: College Admission System using Inner Class

### Problem Statement:
Write a Java program to model a college admission system using the concept of inner classes. Create an outer class named College that stores the collegeName. Define a non-static inner class named Admission that stores studentName and course details. Accept and display student admission details along with the college name.

### Sample Output:

```text
Enter Student Name: Radhe
Enter Course Name: Computer Engineering

Admission Details
College Name: ABC College
Student Name: Radhe
Course: Computer Engineering
```

## Practical 12: Method Overloading for Volume Calculation

### Problem Statement:
Write a Java program that demonstrates method overloading to calculate the volume of different 3D shapes. Implement overloaded methods named calculateVolume() for:
- Cube
- Rectangular Cube
- Sphere

### Sample Output:

```text
Volume of Cube: 125.0
Volume of Rectangular Cube: 120.0
Volume of Sphere: 113.09733552923254
```
